# 城市美食品类与选店指南

## 知识库用途

帮助 Agent 从「菜系类型、场景、口味」角度做科普，与地图 POI 结果配合使用。

## 菜系速览（科普）

### 川菜

麻辣、鲜香、复合味型；常见：麻婆豆腐、回锅肉、火锅。忌重辣者可选微辣或清汤锅。

### 粤菜

偏清淡、原汁原味；常见：白切鸡、煲汤、点心。适合喜鲜、少油人群。

### 湘菜

香辣、咸鲜；与川菜相比更偏鲜辣。注意辣度档位。

### 日料

寿司、刺身、定食；注意生食卫生与过敏（海鲜、芥末）。

### 西餐

牛排、意面、沙拉；注意熟度与生食风险。

### 火锅

适合聚餐；注意汤底盐分、嘌呤（痛风患者遵医嘱）、生熟分开。

### 烧烤

注意烤制火候、油烟；搭配蔬菜，避免过量饮酒。

## 按场景选店（原则）

| 场景 | 建议方向 |
|------|----------|
| 商务简餐 | 出餐快、环境安静、套餐 |
| 家庭聚餐 | 包厢、多品类、儿童友好 |
| 夜宵 | 注意卫生与关闭时间，不宜过饱 |
| 约会 | 环境、服务；提前预订 |

## 与地图 Agent 协作说明

知识库解释「为什么可能适合某类菜系」；具体店名、距离、地址必须来自高德 POI 或用户授权定位后的检索结果，不得凭空列举店名。

## 健康饮食提醒（非医疗）

- 控制油盐糖
- 多样化食材
- 特殊疾病饮食遵医嘱
