# 饮食过敏与就餐急症识别

## 知识库用途

用于 Agent 识别用户描述中的饮食相关高风险信号，并优先给出就医建议。

## 高风险信号（过敏）

用户描述进食后短时间内出现：

- 嘴唇、眼睑肿胀
- 喉咙发紧、声音嘶哑、呼吸困难
- 全身风团、剧烈瘙痒
- 头晕、眼前发黑、血压下降感

### Agent 必须回复要点

1. 立即停止进食可疑食物
2. 严重呼吸困难：拨打 120
3. 已知严重过敏史：提醒携带肾上腺素笔（如有医嘱）
4. 不要继续推荐菜品或地图店铺

## 食物中毒倾向信号

- 多人同时进食后恶心、呕吐、腹泻
- 进食不洁海鲜、隔夜食物、野生菌后不适
- 持续呕吐无法进水

建议：补液、尽快就医；保留可疑食物样本供检验。

## 饮酒相关

- 急性醉酒、意识不清：侧卧防窒息，必要时 120
- 与药物同服风险：提示咨询药师，Agent 不给出具体混饮建议

## 儿童与老人

- 异物卡喉：海姆立克法说明可科普，严重立即 120
- 老人吞咽困难：建议软食、小口慢食，必要时专科评估

## 与推荐流程的关系

一旦识别为 emergency/high，应减少「附近好吃的」营销话术，以安全提示为主；地图 POI 推荐可置后。
